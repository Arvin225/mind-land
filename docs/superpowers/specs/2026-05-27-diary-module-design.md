## 一、概述

在 mind-land 中实现日记模块，替换当前占位页面。采用双栏布局，左侧为现有模块导航栏，内部划分为日记列表（中栏）+ 固定编辑预览区（右栏）。

### 各区域说明

**中栏 — 日记列表**
 年月分组头：中文数字居中对齐（`二零二六年五月`）
 分组时间倒序（最新在上）
 条目左右分栏布局，条目间无边框/分隔线，靠留白区分
 点击条目 → 右侧加载，默认只读模式
 中栏顶部右上角固定新建图标按钮（✏️ 铅笔编辑图标）
**日记条目卡片**

```
┌───────────────────────────────────────┐
│                                       │
│  04 │周  │ 非哲学的人生不值得一过        │
│      │一  │                            │
│  19:14                                │
│                                       │
│  04 │周  │ 奴隶道德是怯弱、懒惰、没有    │
│      │一  │ 个性、循规蹈矩地遵从...      │
│  18:00                                │
│                                       │
│  04 │周  │ 尊重他人，连同他的痛苦，不    │
│      │一  │ 要用你的同情，使他人...      │
│  17:54                                │
│                                       │
└───────────────────────────────────────┘
```

- **左侧时间区**（~70px，右对齐）：
  - 日期数字（如 `04`）：最大字号、加粗
  - 星期几（如 `周一`）：小字号，单独一列，跨越日期/时间两行
    - `周`字与日期同行，`一`字与时间同行
    - 两行对齐后整体呈现“周一”竖排效果
  - 时间（如 `19:14`）：最小字号，日期下方一行，与日期同列

- **右侧摘要区**（flex-1）：
  - 内容摘要，最多 2 行，超长末尾 `...` 截断
  - HTML 转纯文本后渲染
  - 与左侧垂直居中

- **条目间**：无分隔线、无边框，纯靠 padding/margin 留白区分
- **选中态**：左侧日期数字变色 或 整行背景 `--hover` 微变

**右栏 — 编辑 / 预览区**
**右栏 — 编辑 / 预览区**
- 固定区域，不随列表滚动消失
- 从上到下分为三栏：顶部信息栏 → 工具栏 → 内容编辑区

**第一栏：顶部信息栏**

```
┌───────────────────────────────────────────────────┐
│  ⏰ 2026/05/04 17:54  ▼  ● 29字        │  阅读 🗑  │
│  ────────────────────────────────────────────────────┤
│  ↶ ↷ │ ≡ ≡ ≡ │ Aᴀ 16 │ 𝗕 𝑒 𝑼 ̶ │ ≡ │ 🖼       │
│  ────────────────────────────────────────────────────┤
│                                       │
│  尊重他人，连同他的痛苦，不要用你的   │
│  同情，使他人的痛苦平凡化...               │
│                                       │
└───────────────────────────────────────────────────┘
```

- **左侧**：
  - 时钟图标 + `日期时间`（如 `2026/05/04 17:54`）：可点击弹出日期时间选择器修改
  - `● 字数`（如 `29字`）：实时统计 HTML 转纯文本后的字符数
- **右侧**：
  - `阅读 / 编辑` 开关按钮：切换只读与编辑模式
  - 🗑 删除按钮：删除当前日记

**第二栏：工具栏**（按组分隔）**

|分组|功能|
|撤销/重做|撤销、重做|
|对齐缩进|左对齐、居中、右对齐、两端对齐、缩进、减少缩进|
|字体样式|字体颜色、字号、**加粗**、*斜体*、<u>下划线</u>、~~删除线~~|
|列表|有序列表、无序列表|
|媒体|插入图片|

工具栏固定在内容区上方，各组之间用分隔线隔开。

**第三栏：内容编辑区**

- 只读模式：渲染 HTML 内容
- 编辑模式：TipTap 富文本编辑器，可自由输入与排版
- 新建日记自动进入编辑模式；点击已有日记默认只读
- 图片通过现有 `/api/upload` 上传，`<img>` 标签嵌入 HTML

## 三、数据模型

### DiaryEntry（Go / GORM）

```go
type DiaryEntry struct {
    ID        uint      `gorm:"primarykey" json:"id"`
    Content   string    `gorm:"type:text" json:"content"`   // TipTap HTML
    CreatedAt time.Time `json:"createdAt"`
    UpdatedAt time.Time `json:"updatedAt"`
    Del       bool      `json:"del"`                        // 软删除
}
```

- 无心情/天气字段，纯内容型
- 一天可多条，按 `created_at` 排序
- 软删除，删除后不在列表中显示

## 四、API 设计

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/diary/entries` | 分页列表，`?page=1&size=50`，`created_at DESC`，`del=false` |
| GET | `/api/diary/entries/:id` | 单条详情 |
| POST | `/api/diary/entries` | 新建 `{content: "<html>"}` |
| PUT | `/api/diary/entries/:id` | 更新 `{content: "<html>"}` |
| DELETE | `/api/diary/entries/:id` | 软删除 |

**GET /entries 返回格式：**

```json
{
  "code": 0,
  "message": "success",
  "result": {
    "entries": [...],
    "total": 120,
    "page": 1,
    "size": 50
  }
}
```

条目按 `created_at DESC` 排序，前端本地按年→月→日分组渲染。

### 组件行为

| 组件 | 职责 |
|------|------|
| `Diary/index.tsx` | flex 容器，中栏 ~360px，右栏 flex-1。加载时 `fetchGetEntries()` |
| `DiaryList.tsx` | 接收 `entries[]`，按年→月→日分组，渲染 `<YearGroup>` / `<MonthGroup>` / `<DiaryCard>` |
| `DiaryCard.tsx` | 显示 `HH:mm` + 纯文本摘要。点击 `onClick(id)` → store 设 `selectedId`，右栏加载只读 |
| `DiaryEditor.tsx` | 读写开关控制。编辑模式激活 TipTap；只读模式渲染 `dangerouslySetInnerHTML`。新建/编辑后刷新列表 |

### 日期分组格式化

年月头用中文数字居中，日期时间用阿拉伯数字：

```typescript
const CN_DIGITS = ['零','一','二','三','四','五','六','七','八','九'];

function toChineseYearMonth(year: number, month: number): string {
  const y = String(year).split('').map(d => CN_DIGITS[+d]).join('');
  return `${y}年${month}月`;  // 如：二零二六年五月
}

// 分组：年月头居中，条目显示 "DD日 HH:mm"
// 二零二六年五月         ← 居中对齐
//   27日 09:30           ← 阿拉伯数字
//   26日 14:20
```

## 七、图片处理

日记中插入的图片通过现有 `/api/upload` 接口上传：

- TipTap 编辑器配置图片上传 handler → `POST /api/upload`
- 返回 `{code:0, result: {url: "/uploads/<uuid>.jpg"}}`
- 图片作为 `<img src="/uploads/<uuid>.jpg">` 嵌入 HTML 内容
- 后端已有 `r.Static("/uploads", "./uploads")`，无需额外配置

## 八、与现有系统关系

- **编辑器：** TipTap，diary 有独立的 ToolBar 配置（见上方工具栏表格），不直接复用 SlipBox 的编辑器配置
- **复用图片上传：** 使用现有 `/api/upload`
- **遵循模式：** 后端 model/service/handler，前端 Redux Toolkit + axios
- **不修改现有模块：** Diary 完全独立，不改动 SlipBox/ToDo/其他页面
- **路由已有：** `/diary` 路由已在 `router/index.tsx` 中注册，只需替换页面组件
